Bukkitmanager by efreak1996

Bukkitmanager is the plugin you have waited for! It manages and updates the whole Bukkitserver with Spout support, Webinterface and Swing-GUI.

Install

To install Bukkitmanager you have only to copy the bukkitmanager.jar into the plugins Folder of your bukkitserver. Bukkitmanager will create its needed stuff by itself
